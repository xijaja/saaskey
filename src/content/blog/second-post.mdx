# 进阶排版：代码块、引用与表格

这篇文章用于演示 **代码块、引用、表格** 等内容在博客中的展示效果。

## 代码块示例

```ts
import { allBlogPosts } from "content-collections";

export function getPublishedPosts() {
  return allBlogPosts.filter((post) => post.published !== false);
}
```

> 提示：代码块支持高亮，适合用来展示配置片段、API 示例等技术内容。

## 表格示例

| 功能           | 说明                                   |
| -------------- | -------------------------------------- |
| Markdown 支持  | 标准 Markdown + GFM                    |
| MDX 组件       | 可以在文章中直接引入 React 组件       |
